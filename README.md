# Pingbase

The calm alternative to Slack. Built with Elixir and Phoenix LiveView.

## Features

- Multi-tenant team chat
- Self-hosted = all features free
- SaaS = per-seat freemium (10 free users)
- Slack-compatible webhooks
- MCP server support (v2)
- CLI support (v2)

## Development

```bash
# Setup
mix setup

# Run dev server
mix phx.server

# Run tests
mix test
```
