# Pingbase

An open-source, multi-tenant team chat application — the calm alternative to Slack.

## Philosophy

- **Calm over chaos**: No notification overload, no read receipts, no green-dot anxiety
- **User in control**: Opt-out notifications, focus mode, scheduled send
- **Simple over complex**: Small tested core, no AI bloat, no feature creep
- **Freedom**: Self-hosted = all features free forever

## Tech Stack

- **Backend**: Elixir / Phoenix / LiveView
- **Frontend**: Tailwind CSS + DaisyUI
- **Database**: PostgreSQL
- **Jobs**: Oban
- **Auth**: Magic links + API tokens
- **Billing**: Stripe (SaaS only)
- **Files**: S3-compatible storage
